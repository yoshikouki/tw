# tw

## Installation

Require to setup deno.

https://deno.land/

```sh
$ git clone https://github.com/yoshikouki/tw.git && cd tw && make install
```

## Usage

### tweet #WIP

```sh
$ tw tweet by tw!!1
```

### Check Timeline #WIP

```sh
$ tw -l
```

### Change Twitter Account #WIP

```sh
# change twitter account
$ tw --account tw_test

# show register account list
$ tw --accounts
```
